static int8_t get_calib_data ( struct bme680_dev *  dev)
static

This internal API is used to read the calibrated data from the sensor.

This function is used to retrieve the calibration data from the image registers of the sensor.

Note
Registers 89h to A1h for calibration data 1 to 24 from bit 0 to 7.
Registers E1h to F0h for calibration data 25 to 40 from bit 0 to 7.
Parameters
[in]dev:Structure instance of bme680_dev
Returns
Result of API execution status
Return values
zero-> Success / +ve value -> Warning / -ve value -> Error
